    COMANDOS DISPONÍVEIS

    👤 PARA TODOS OS USUÁRIOS:

    /pm <usuário> <msg> - Mensagem privada
    /togglepm - Ativa/desativa mensagens privadas
    /votekick <usuário> - Votação para expulsar
    /votemute <usuário> [min] - Votação para mutar
    /sim, /não - Votar em votações ativas
    /emojis - Mostrar emojis disponíveis
    /delete <id> - Deletar sua mensagem
    /help - Mostrar esta ajuda

    👑 ADMIN/DONO DA SALA:

    /kick <usuário> [razão] - Expulsar usuário
    /warn <usuário> [razão] - Advertir usuário
    /mute <usuário> [min] [razão] - Mutar usuário
    /unmute <usuário> - Desmutar usuário
    /broadcast <msg> - Mensagem para todos
    /delete <id> - Deletar qualquer mensagem
